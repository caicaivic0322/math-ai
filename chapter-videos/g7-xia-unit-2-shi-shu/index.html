知识点 1：平方根

平方根：什么数平方后等于它？

求平方根不是背答案，而是反问逆运算。49 的平方根有两个：7 和 -7。

?² = 49
→
7² = 49
(-7)² = 49
→
±7

平方根

±√49 = ±7
包含正负两个结果

算术平方根

√49 = 7
只取非负的那一个

负数没有平方根。0 的平方根是 0。
知识点 2：立方根

立方根：x³ = a → x = ∛a

立方根和平方根不同——负数也有立方根。比如 (-3)³ = -27。

(-3)³ = -27
→
∛(-27) = -3

平方根

负数没有平方根 ✗

立方根

负数有立方根 ✓

做题先看清楚：平方根、算术平方根，还是立方根？
知识点 3：实数

有理数 + 无理数 = 实数

有限小数和无限循环小数是有理数；无限不循环小数是无理数。它们统称实数。

不是所有带根号的数都是无理数！√9 = 3 是有理数。
知识点 4：数轴估算

用平方关系给根号数定位

实数都能在数轴上找到对应点。用"夹逼"法不断缩小范围。

1
2²=4 < 5 < 9=3² → 2 < √5 < 3
2
2.2²=4.84, 2.3²=5.29 → 2.2 < √5 < 2.3
2.2 < √5 < 2.3
