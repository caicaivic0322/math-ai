# 七年级上册 第三章 一元一次方程 学习视频脚本

## 旁白稿

同学你好，这一节我们学习七年级上册第三章，一元一次方程。

这一章会帮你把文字问题翻译成方程，再把方程一步一步解出来。我们重点抓三件事：什么是方程，怎样检验方程的解，以及怎样解一元一次方程。

先看方程。含有未知数的等式叫做方程。比如“一个数加五等于十二”，设这个数为 x，就可以写成 x + 5 = 12。注意，只有式子含有未知数还不够，它还必须是等式。

什么叫一元一次方程？只含有一个未知数，并且未知数的次数是 1 的方程，叫做一元一次方程。比如 3x = 18、2x - 7 = 9，都是一元一次方程。但 x + y = 10 有两个未知数，x² = 9 中未知数的次数不是 1，所以都不符合。

方程的解，是让左右两边相等的未知数的值。检验时，把这个数代入原方程的左边和右边分别计算。例如 x + 5 = 12，代入 x = 7，左边是 7 + 5 = 12，右边也是 12，所以 7 是这个方程的解。只看算出来像不像答案是不够的，代回原方程才稳。

接下来学习解一元一次方程。我们的目标，是把方程变形成 x = a 这样的形式。常用顺序是：去括号、移项、合并同类项、系数化为 1。每一步都依据等式性质，方程两边同时做同样的运算，等式仍然成立。

看第一个例题，文具费用。一本练习本 3 元，又买一支 5 元的笔，一共花 20 元，问买了几本练习本。设买了 x 本，练习本费用就是 3x 元。等量关系是：练习本费用加笔的费用，等于总费用，所以列方程 3x + 5 = 20。移项得 3x = 15，两边同时除以 3，得到 x = 5。检验一下，3×5+5=20，所以答案是 5 本。

再看一个未知数在两边的例题。左边有 2x + 4 个座位，右边有 x + 16 个座位，左右相同。列出方程：2x + 4 = x + 16。把右边的 x 移到左边，把左边的 4 移到右边，注意移项要变号。得到 2x - x = 16 - 4，所以 x = 12。

本章最容易错的地方有三个。第一，把不是等式的式子也当成方程；第二，忘记检查未知数个数和次数；第三，移项只移动位置却忘记变号。解完以后，把答案代回原方程检验，能帮你抓住很多符号错误。

最后记住一条主线：找等量关系，列方程，按步骤求解，再代回检验。学完这一章，你要能从实际问题里设未知数、列方程，并稳稳地解出一元一次方程。现在回到软件里完成第三章单元小测，看看自己能不能把方程用起来。
