同学你好，这一节我们学习七年级上册第三章，一元一次方程。
这一章会帮你把文字问题翻译成方程，再把方程一步一步解出来。
我们重点抓三件事：什么是方程，怎样检验方程的解，以及怎样解一元一次方程。
先看方程，含有未知数的等式叫做方程。
比如一个数加五等于十二，设这个数为 x，就可以写成 x 加五等于十二。
注意，只有式子含有未知数还不够，它还必须是等式。
什么叫一元一次方程？只含有一个未知数，并且未知数的次数是 1 的方程，叫做一元一次方程。
比如三 x 等于十八、二 x 减七等于九，都是一元一次方程。
但 x 加 y 等于十有两个未知数，x 平方等于九次数不是一，所以都不符合。
方程的解，是让左右两边相等的未知数的值。
检验时，把这个数代入原方程的左边和右边分别计算。
例如 x 加五等于十二，代入 x 等于七，左边是七加五等于十二，右边也是十二，所以七是这个方程的解。
只看算出来像不像答案是不够的，代回原方程才稳。
接下来学习解一元一次方程，我们的目标，是把方程变形成 x 等于某个数。
常用顺序是：去括号、移项、合并同类项、系数化为一。
每一步都依据等式性质，方程两边同时做同样的运算，等式仍然成立。
看第一个例题，文具费用：一本练习本三元，又买一支五元的笔，一共花二十元，问买了几本练习本。
设买了 x 本，练习本费用就是三 x 元。
等量关系是：练习本费用加笔的费用，等于总费用，所以列方程三 x 加五等于二十。
移项得三 x 等于十五，两边同时除以三，得到 x 等于五。
检验一下，三乘五再加五等于二十，所以答案是五本。
再看一个未知数在两边的例题：左边有二 x 加四个座位，右边有 x 加十六个座位，左右相同。
列出方程：二 x 加四等于 x 加十六。
把右边的 x 移到左边，把左边的四移到右边，注意移项要变号。
得到二 x 减 x 等于十六减四，所以 x 等于十二。
本章最容易错的地方有三个。第一，把不是等式的式子也当成方程；第二，忘记检查未知数个数和次数。
第三，移项只移动位置却忘记变号。
解完以后，把答案代回原方程检验，能帮你抓住很多符号错误。
最后记住一条主线：找等量关系，列方程，按步骤求解，再代回检验。学完这一章，你要能从实际问题里设未知数、列方程，并稳稳地解出一元一次方程。
