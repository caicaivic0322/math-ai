目标：把方程变形成 x = a

例题 1：文具费用

先设未知数，再列方程

练习本 3 元一本，又买一支 5 元的笔，总共花 20 元。求买了几本练习本。

设买了 x 本，练习本费用是 3x 元
1
等量关系：练习本费用 + 笔的费用 = 总费用。
2
列方程：3x + 5 = 20。
3
移项：3x = 15；两边除以 3，x = 5。
4
检验：3×5+5=20，所以答案是 5 本。
例题 2：未知数在两边

移项要变号

左右两边座位数相同，把含 x 的项放到一边，常数项放到另一边。

左边：2x + 4　右边：x + 16
1
列方程：2x + 4 = x + 16。
2
右边的 x 移到左边，左边的 4 移到右边。
3
移项变号：2x - x = 16 - 4。
4
合并后得到 x = 12。
移项不是搬位置，是两边同时加减后的简写，所以符号要改变。

先看方程：含有未知数的等式叫做方程。
一个数加 5 等于 12，可以写成 x + 5 = 12。
只有未知数还不够，它还必须是等式。
一元一次方程：一个未知数，次数是 1。
3x = 18 和 2x - 7 = 9 都是一元一次方程。
x + y = 10 和 x² = 9 都不符合。
方程的解：让左右两边相等的未知数的值。
检验要代回原方程，左右两边分别计算。
x = 7 时，7 + 5 = 12，左右相等。
只看答案不够，代回原方程才稳。
解方程目标：把方程变形成 x = a。
常用顺序：去括号、移项、合并同类项、系数化为 1。
等式性质：两边同时做同样运算，等式仍然成立。
例题：一本练习本 3 元，一支笔 5 元，总共 20 元。
设买了 x 本，练习本费用就是 3x 元。
练习本费用 + 笔的费用 = 总费用，列方程 3x + 5 = 20。
移项得 3x = 15，两边除以 3，得到 x = 5。
检验：3×5+5=20，所以答案是 5 本。
例题：左边 2x + 4，右边 x + 16，左右相同。
列出方程：2x + 4 = x + 16。
移项要变号：右边的 x 到左边，左边的 4 到右边。
得到 2x - x = 16 - 4，所以 x = 12。
易错点：不是等式也当方程；忘记检查未知数个数和次数。
