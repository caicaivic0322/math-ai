知识点 1：相交线

邻补角看公共边，对顶角看面对面

两条直线相交形成四个角。判断前先看两个角之间有没有公共边。

邻补角

相邻、有公共边，和为 180°。

对顶角

面对面：∠1 = ∠3，∠2 = ∠4。

如：∠1 + ∠2 = 180°
邻补角互补，对顶角相等。

先找到截线，再判断角的位置

图中八个角都先标出来，再说同位、内错、同旁内，位置就不会乱。

截线 c：同时穿过 a 和 b

同位角

例：∠2 和 ∠6，位置上下对应。

同旁内角

例：∠4 和 ∠6，在内侧且同在截线右侧。

知识点 3：性质与判定

已知平行求角，已知角判平行

同一组角关系可以正着用，也可以反着用。先看题目给了什么。

性质
a ∥ b → 同位角相等，内错角相等
性质
a ∥ b → 同旁内角互补
判定
同位角相等 或 内错角相等 → a ∥ b
判定
同旁内角互补 → a ∥ b
先判断方向：题目给“平行”，还是让你证明“平行”？
例题 1：相交线求角

一个角 58°，邻补角和对顶角分别是多少？

四个角先都标清楚：已知角、邻补角、对顶角各在哪里。

1
邻补角与 58° 组成平角：180° - 58° = 122°。
2
对顶角与已知角相等：对顶角 = 58°。
答案：邻补角 122°，对顶角 58°
例题 2：平行线角度追踪

a ∥ b，已知 ∠4 = 68°，追踪对应角

先把八个角标完，再找同位角和同旁内角，避免把角看错。

1
∠4 和 ∠8 是同位角，所以 ∠8 = 68°。
2
∠4 和 ∠6 是同旁内角，所以 ∠6 = 180° - 68° = 112°。
答案：∠8 = 68°，∠6 = 112°

平行线题看起来角很多，但先找截线，局面会清楚很多。

这一节学习：七年级下册第一章，相交线与平行线。
先学邻补角和对顶角，再学平行线中的三类角。
主线：找线和截线，定角的位置，用相等或互补计算。
两条直线相交形成四个角，先看有没有公共边。
邻补角：有公共边，另外两边在同一直线上，和为 180°。
对顶角：面对面的两个角，对顶角相等。
邻补角互补，对顶角相等，别混淆。
平行线问题，先找到同时穿过两条直线的截线。
同位角：两条线同侧，截线同侧，位置上下对应。
内错角：两条线之间，分居截线两侧。
同旁内角：两条线之间，在截线同一侧。
两直线平行：同位角相等，内错角相等，同旁内角互补。
同位角或内错角相等，可以判定两直线平行。
同旁内角互补，也可以判定两直线平行。
先看题目给的是“平行”还是“角度关系”。
例题一：已知一个角 58°，求邻补角和对顶角。
邻补角 = 180° - 58° = 122°。
对顶角与已知角相等，所以是 58°。
答案：邻补角 122°，对顶角 58°。
例题二：a ∥ b，截线 c 所截，已知 ∠4 = 68°。
同位角相等：∠8 = ∠4 = 68°。
同旁内角互补：∠6 = 180° - 68° = 112°。
答案：∠8 = 68°，∠6 = 112°。
易错点 1：没先找截线就判断角名。
易错点 2：把同旁内角误认为相等；易错点 3：漏写单位。
三步：找截线，定位置，用关系。
相交线看邻补和对顶，平行线看三类角。
会识别角关系，会用相等和互补求角，也能判定平行。
