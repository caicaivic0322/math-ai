# 七年级下册 第一章 相交线与平行线 学习视频脚本

## 旁白稿

同学你好，这一节我们学习七年级下册第一章，相交线与平行线。

本章先看两条直线相交形成的邻补角和对顶角，再看平行线被截线截出的三类角。学习主线是：先找线和截线，再定角的位置，最后用相等或互补关系计算。

两条直线相交，会形成四个角。判断角关系，先看是否有公共边。有公共顶点和一条公共边，另外两边在同一直线上，这两个角是邻补角，和是 180°。面对面的两个角是对顶角，对顶角相等。易混处是：邻补角不是相等，而是互补；对顶角才相等。

平行线问题里，要先找到截线。截线是同时穿过两条直线的那条线。同位角在两条直线的同一侧，也在截线的同一侧，位置像上下对应。内错角在两条直线之间，并且分居截线两侧。同旁内角也在两条直线之间，但它们在截线同一侧。

如果两直线平行，那么同位角相等，内错角相等，同旁内角互补。反过来，如果同位角相等，或者内错角相等，就可以判定两条直线平行；如果同旁内角互补，也可以判定两条直线平行。做题时先判断题目给的是平行还是角度关系，方向不要用反。

看例题一：两条直线相交，已知其中一个角为 58°，求它的邻补角和对顶角。邻补角与已知角组成平角，所以邻补角等于 180° - 58° = 122°。对顶角与已知角相等，所以对顶角等于 58°。答案是：邻补角 122°，对顶角 58°。

看例题二：直线 a ∥ b，被截线 c 所截，在点 A 处有一个内角是 68°，也就是 ∠4 = 68°。与 ∠4 同位的角是 ∠8，因为两直线平行，所以 ∠8 也等于 68°。与 ∠4 同旁内的角是 ∠6，它们互补，所以 ∠6 等于 180° - 68° = 112°。答案是：∠8 = 68°，∠6 = 112°。

本章最容易错的地方有三个。第一，没先找截线，就急着判断角名。第二，把同旁内角误认为相等；第三，知道要减 180°，却漏写角度单位。每次做题按三步来：找截线，定位置，用关系。

最后总结：相交线看邻补和对顶，平行线看同位、内错和同旁。学完这一章，你要能识别角关系，会用相等和互补求角，也能根据角关系判定平行。现在回到软件里完成第一章单元小测，检查自己能不能把图形中的关系读出来。

4. 两条直线相交形成四个角，先看有没有公共边。
5. 邻补角：有公共边，另外两边在同一直线上，和为 180°。
6. 对顶角：面对面的两个角，对顶角相等。
7. 邻补角互补，对顶角相等，别混淆。
8. 平行线问题，先找到同时穿过两条直线的截线。
9. 同位角：两条线同侧，截线同侧，位置上下对应。
10. 内错角：两条线之间，分居截线两侧。
11. 同旁内角：两条线之间，在截线同一侧。
12. 两直线平行：同位角相等，内错角相等，同旁内角互补。
13. 同位角或内错角相等，可以判定两直线平行。
14. 同旁内角互补，也可以判定两直线平行。
15. 先看题目给的是“平行”还是“角度关系”。
16. 例题一：已知一个角 58°，求邻补角和对顶角。
17. 邻补角 = 180° - 58° = 122°。
18. 对顶角与已知角相等，所以是 58°。
19. 答案：邻补角 122°，对顶角 58°。
20. 例题二：a ∥ b，截线 c 所截，已知 ∠4 = 68°。
21. 同位角相等：∠8 = ∠4 = 68°。
22. 同旁内角互补：∠6 = 180° - 68° = 112°。
23. 答案：∠8 = 68°，∠6 = 112°。
24. 易错点 1：没先找截线就判断角名。
25. 易错点 2：把同旁内角误认为相等；易错点 3：漏写单位。
26. 三步：找截线，定位置，用关系。
27. 相交线看邻补和对顶，平行线看三类角。
