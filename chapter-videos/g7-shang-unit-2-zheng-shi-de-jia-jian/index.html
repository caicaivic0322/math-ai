知识点 1：同类项

先看字母部分
是不是完全一样

判断同类项时，关键不在系数，而在字母种类和相同字母的指数是不是完全一致。

是同类项

3x 和 -5x

2a²b 和 -7a²b

不是同类项

4ab 和 4a

x² 和 x³

有相同字母还不够，指数不同，也不能算同类项。
易错判断

字母不同
指数不同，都不能合并

别被“都有一个字母”骗了。同类项要求字母部分完全一致。

4ab 和 4a：不是同类项
x² 和 x³：不是同类项
看到同一个字母就急着合并，是这一章最常见的失分点之一。
合并规则

只合并系数
字母部分不变

同类项能合并，不是同类项就不能硬凑。处理时，只对前面的数字做加减。

3x + 5x = 8x
7a²b - 2a²b = 5a²b
4m + 2n 不能合并
- ( )
知识点 2：整式的加减

先去括号，再合并同类项

加号在前

符号不变

括号里的每一项照原样写出

a + (2a - 3)
减号在前

每一项都变号

不能只改第一项，后面的常数也要改

x - (3x + 2)
标准顺序

去括号

然后再找同类项，最后完成化简

先去括号 → 再合并
例题 1：合并同类项

先找同类项
再分别合并

化简：4x + 7 - 2x + 3。这里的关键是，不要把字母项和常数项混在一起算。

原式：4x + 7 - 2x + 3
1
4x 和 -2x 是同类项，7 和 3 也是同类项。
2
4x - 2x = 2x，7 + 3 = 10。
3
所以原式化简为 2x + 10。
例题 2：去括号化简

括号前是减号
每一项都要变号

化简：3a - (2a - 5) + 4。只要括号前有减号，里面每一项都要一起改符号。

原式：3a - (2a - 5) + 4
去括号后：3a - 2a + 5 + 4
合并后：a + 9
最常见错误：只把 2a 变号，却忘了把 -5 变成 +5。
本章易错点

这两个地方
最容易失分

很多题并不难，错就错在判断顺序乱了，或者处理括号时漏了一项。

1
把不是同类项的项硬合并
2
括号前有减号，却漏掉后面项的变号
3
没有按“先去括号，再合并”的顺序做题
先看字母部分 → 再看括号前符号 → 最后合并

这一节学习：七年级上册第二章，整式的加减。
这一章主要解决两件事。
第一，什么叫同类项，哪些项能合并。
第二，遇到括号以后，怎样按顺序完成整式的加减化简。
先看同类项。
判断同类项：先看字母部分是不是完全一样。
3x 和 -5x 是同类项；2a²b 和 -7a²b 也是同类项。
因为字母相同，相同字母的指数也一样。
字母不同，或者指数不同，就不是同类项。
4ab 和 4a 不是同类项；x² 和 x³ 也不是同类项。
最容易丢分：看到同一个字母，就急着合并。
接下来记住合并规则。
合并同类项：只合并系数，字母和指数保持不变。
3x + 5x = 8x；7a²b - 2a²b = 5a²b。
不是同类项，不能硬合并。
再看整式的加减。
最稳顺序：先去括号，再合并同类项。
括号前是加号：括号内各项符号不变。
括号前是减号：每一项都要变号，不能只改第一项。
看第一个例题。
化简：4x + 7 - 2x + 3。
4x 和 -2x 是同类项，7 和 3 也是同类项。
4x - 2x = 2x，7 + 3 = 10，所以结果是 2x + 10。
再看第二个例题。
化简：3a - (2a - 5) + 4。
括号前是减号：2a 变成 -2a，-5 变成 +5。
原式变成：3a - 2a + 5 + 4。
3a - 2a = a，5 + 4 = 9，所以结果是 a + 9。
这一章最关键的失分点有两个。
第一，把不是同类项的项硬合并。
第二，去括号时漏变号，尤其漏掉后面的常数项。
口诀：先看字母部分，再看括号前符号，最后合并同类项。
学完这一章：会判断同类项，会去括号，会一步一步化简整式。
