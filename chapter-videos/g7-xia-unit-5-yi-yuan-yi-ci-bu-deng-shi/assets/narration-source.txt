同学你好，这一节我们学习七年级下册第五章，一元一次不等式与一元一次不等式组。

这一章会把方程思想继续向前推进。方程常常是求一个确定的值，而不等式更多是在找一段范围。学习主线可以记成四句话：先看大小关系，再求解集；数轴表示看端点；遇到负数运算要变号；不等式组要取公共部分。

先看不等式。用大于、小于、大于等于、小于等于表示大小关系的式子，都可以是不等式。比如 x 大于 3，表示所有比 3 大的数；x 小于等于负 1，表示负 1 以及比负 1 更小的数。

能使不等式成立的未知数的值，叫做这个不等式的解。所有解合在一起，叫做解集。这里和方程很不一样：方程 x + 2 = 5 的解是 x = 3；不等式 x > 3 的解不是一个数，而是所有大于 3 的数。

把解集画在数轴上时，要特别看端点。大于或小于，用空心点，表示端点不包含。大于等于或小于等于，用实心点，表示端点包含。箭头方向也要看清：x > 4 向右，x ≤ -2 向左。

接下来解一元一次不等式。前面的步骤和解方程很像：移项、合并同类项、系数化为 1。但有一个关键区别：不等式两边同乘或同除以同一个负数时，不等号方向要改变。比如 -2x > 6，两边同除以 -2，得到 x < -3。这里如果忘记变号，答案方向就会完全相反。

看例题一：解不等式 3x - 5 > 7，并在数轴上表示解集。先把 -5 移到右边，得到 3x > 12。再两边同除以 3，得到 x > 4。因为是不包含 4 的“大于”，所以在数轴上 4 处画空心点，向右画射线。

再看不等式组。一个不等式组里有多个条件，最后的解集必须同时满足所有条件。所以做题时要分别求出每个范围，再取公共部分。比如 x > -1，x ≤ 3，公共部分就是 -1 < x ≤ 3。负 1 不包含，要空心；3 包含，要实心。

看例题二：解不等式组 x > -1，x ≤ 3。第一个条件要求 x 在 -1 的右边，第二个条件要求 x 不超过 3。两个范围重叠的部分，是从 -1 到 3，其中不包含 -1，包含 3，所以答案是 -1 < x ≤ 3。

本章最容易错的地方有三个。第一，把不等式的解集误写成一个等号答案。第二，数轴端点空心和实心画反。第三，同乘或同除以负数时忘记改变不等号方向。遇到不等式组，还要提醒自己：不是把两个范围并起来，而是取公共部分。

最后总结：不等式看范围，数轴看端点，负数运算要变号，不等式组取公共部分。学完这一章，你要能从文字条件列出不等式，会解一元一次不等式，也能在数轴上表示解集。现在回到软件里完成第五章单元小测，检查自己能不能把范围读准、算准、画准。
