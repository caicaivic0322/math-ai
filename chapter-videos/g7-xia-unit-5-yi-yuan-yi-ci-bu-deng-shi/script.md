# 七年级下册 第五章 一元一次不等式与一元一次不等式组 学习视频脚本

## 旁白稿

同学你好，这一节我们学习七年级下册第五章，一元一次不等式与一元一次不等式组。

这一章会把方程思想继续向前推进。方程常常是求一个确定的值，比如 x 等于几；不等式更多是在找一段范围，比如 x 大于几，或者 x 落在两个数之间。所以看到不等式时，先不要急着写等号答案。学习主线可以记成四句话：先看大小关系，再求解集；数轴表示看端点；遇到负数运算要变号；不等式组要取公共部分。

先看不等式。用大于、小于、大于等于、小于等于表示大小关系的式子，都可以是不等式。这里的符号不是装饰，它直接告诉我们哪些数能取、哪些数不能取。比如 x 大于 3，不是说 x 只能等于 4，而是说 3 右边的数都可以，像 3.1、4、10 都满足。x 小于等于负 1，表示负 1 本身可以取，负 2、负 3 这些更小的数也可以取。

能使不等式成立的未知数的值，叫做这个不等式的解。判断一个数是不是解，最直接的方法就是代进去，看大小关系是否成立。所有解合在一起，叫做解集。这里和方程很不一样：方程 x + 2 = 5 的解是 x = 3，因为只有 3 代进去刚好相等；不等式 x > 3 的解不是一个数，而是所有大于 3 的数，答案是一整段范围。

把解集画在数轴上时，要特别看端点。端点就是分界的那个数，比如 x > 4 里的 4，x ≤ -2 里的 -2。大于或小于没有等号，用空心点，表示端点不包含。大于等于或小于等于有等号，用实心点，表示端点包含。箭头方向也要看清：x > 4 表示比 4 大，向右；x ≤ -2 表示 -2 和它左边的数，向左。

接下来解一元一次不等式。前面的步骤和解方程很像：先移项，把含 x 的项和常数项分开；再合并同类项；最后把 x 的系数化为 1。但有一个关键区别：不等式两边同乘或同除以同一个负数时，大小顺序会反过来，所以不等号方向要改变。比如 -2x > 6，两边同除以 -2，得到 x < -3。这里如果忘记变号，答案方向就会完全相反。

看例题一：解不等式 3x - 5 > 7，并在数轴上表示解集。先看结构，左边有含 x 的 3x，也有常数 -5；目标是把 x 单独留下。第一步，把 -5 移到右边，也就是两边同时加 5，得到 3x > 12。第二步，两边同除以正数 3，不等号方向不变，得到 x > 4。因为 4 不包含在解集里，所以在数轴上 4 处画空心点；比 4 大的数在右边，所以向右画射线。

再看不等式组。一个不等式组里有多个条件，最后的解集必须同时满足所有条件。注意“同时”这两个字，只满足其中一个条件还不够。做题时先把每个不等式分别画成一个范围，再看两段范围重叠在哪里。重叠的那一段，才是不等式组的公共部分。比如 x > -1，x ≤ 3，公共部分就是 -1 < x ≤ 3。负 1 不包含，要空心；3 包含，要实心。

看例题二：解不等式组 x > -1，x ≤ 3。第一个条件要求 x 在 -1 的右边，第二个条件要求 x 不超过 3。两个范围重叠的部分，是从 -1 到 3，其中不包含 -1，包含 3，所以答案是 -1 < x ≤ 3。

本章最容易错的地方有三个。第一，把不等式的解集误写成一个等号答案。第二，数轴端点空心和实心画反。第三，同乘或同除以负数时忘记改变不等号方向。遇到不等式组，还要提醒自己：不是把两个范围并起来，而是取公共部分。

最后总结：不等式看范围，数轴看端点，负数运算要变号，不等式组取公共部分。学完这一章，你要能从文字条件列出不等式，会解一元一次不等式，也能在数轴上表示解集。现在回到软件里完成第五章单元小测，检查自己能不能把范围读准、算准、画准。
