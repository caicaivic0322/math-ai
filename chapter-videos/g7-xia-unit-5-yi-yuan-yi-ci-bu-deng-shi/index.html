知识点 1：不等式与解集

方程找一个值，不等式找一段范围

能使不等式成立的未知数的值叫做解；所有解组成解集。

x > 3

所有比 3 大的数。

x ≤ -1

-1 以及比 -1 更小的数。

-2 < x ≤ 4

一段带端点条件的范围。

x + 2 = 5 → x = 3
x > 3 → 所有大于 3 的数
知识点 2：数轴表示

端点决定空心还是实心

大于、小于不包含端点；大于等于、小于等于包含端点。箭头方向表示范围方向。

x > 4：空心点，向右

x ≤ -2：实心点，向左

知识点 3：解一元一次不等式

像解方程一样化简，但要盯住不等号

移项、合并同类项、系数化为 1。只有遇到同乘或同除以负数时，不等号方向必须改变。

步骤
移项 → 合并同类项 → 系数化为 1
普通运算
两边同加、同减、同乘正数，方向不变
负数运算
两边同乘或同除以负数，方向改变
例：-2x > 6，两边除以 -2，得到 x < -3。

知识点 4：不等式组

多个条件，要同时满足

分别求出每个不等式的范围，再在数轴上留下重叠的公共部分。

x > -1
x ≤ 3
公共部分：-1 < x ≤ 3
例题 2：解不等式组

x > -1，x ≤ 3

第一个条件要求在 -1 右边，第二个条件要求不超过 3。答案是两个范围的交集。

-1 不包含画空心，3 包含画实心。
答案：-1 < x ≤ 3

学习七年级下册第五章：一元一次不等式与一元一次不等式组。这一章学会把条件翻译成范围，再准确地写出来、画出来。
方程常常求一个确定的值，比如 x 等于几。不等式更多是在找一段范围，比如 x 大于几。所以看到不等式，先不要急着写等号答案。
主线四步：先看大小关系，再求解集；数轴要看端点；遇到负数运算要变号；不等式组则要取公共部分。
不等式用大于、小于、大于等于、小于等于表示大小关系。这里的符号不是装饰，它直接告诉我们哪些数能取、哪些数不能取。
比如 x 大于 3，不是说 x 只能等于 4，而是 3 右边的数都可以，像 3.1、4、10 都满足条件。
x 小于等于负 1，表示负 1 本身可以取，负 2、负 3 这些更小的数也可以取。有没有等号会影响端点是否包含。
能使不等式成立的未知数的值，叫做不等式的解。判断方法很简单：把数代进去，看大小关系是否成立。
所有解合在一起，叫做解集。解集通常是一段范围，最后要写成 x 大于几、x 小于等于几的形式。
对比一下：方程 x 加 2 等于 5 的解只有 x 等于 3，因为只有 3 代进去刚好相等。
但不等式 x 大于 3 的解集是所有大于 3 的数。答案不是一个点，而是从 3 往右的一整段，这是方程和不等式最大的区别。
数轴表示解集时，先看端点是否包含。端点是分界的那个数，比如 x 大于 4 里的 4，x 小于等于负 2 里的负 2。
大于或小于没有等号，画空心点，表示端点不包含。大于等于或小于等于有等号，画实心点，表示端点包含。
方向也要一起看。x 大于 4，表示比 4 大的数，箭头向右。x 小于等于负 2，表示负 2 和它左边的数，箭头向左。
解一元一次不等式，前半段和解方程很像：先移项，把含 x 的项和常数项分开；再合并同类项；最后把系数化为 1。
关键区别：不等式两边同乘或同除以同一个负数时，大小顺序会反过来，所以不等号方向必须改变。
例如负 2x 大于 6。为了得到 x，两边除以负 2。因为除的是负数，大于号变成小于号，得到 x 小于负 3。
例题一：解不等式 3x 减 5 大于 7。左边有含 x 的 3x 和常数负 5，目标是单独留下 x。
第一步把负 5 移到右边，两边同时加 5，得 3x 大于 12。第二步两边同除以正数 3，不等号方向不变，得到 x 大于 4。
数轴表示时，4 只是分界点，不包含在解集里，所以画空心点。比 4 大的数在右边，向右画射线。
不等式组的解集必须同时满足所有条件。注意同时这两个字，只满足其中一个条件还不够，两个条件都满足的数才留下。
先把每个不等式分别画成一个范围，再看两段范围重叠在哪里。重叠的那一段，才是不等式组的公共部分。
例题二：x 大于负 1，x 小于等于 3。第一个条件要求 x 在负 1 的右边，第二个条件要求 x 不超过 3。
两个范围重叠的部分从负 1 的右边开始，到 3 结束。所以解集写作负 1 小于 x，且 x 小于等于 3。
端点也要对应准确：负 1 不包含，画空心；3 包含，画实心。图和式子必须一一对应。
易错点一：把解集写成一个等号答案。看到不等式时要提醒自己，答案通常是一段范围，不是 x 等于几。
易错点二：空心点和实心点画反。判断方法很简单：有等号就包含端点，画实心；没有等号就不包含，画空心。
易错点三：除以负数时忘记变号。只要最后一步是除以负数，先停一下，检查不等号有没有反向。
最后记住口诀：不等式看范围，数轴看端点，负数运算要变号。不等式组还要加一句：多个条件取公共部分。
